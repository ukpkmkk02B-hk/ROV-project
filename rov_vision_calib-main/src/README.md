# src 源码模块说明

本目录保存可复用的功能代码，脚本只负责解析命令行和组织流程。

## 子目录

- `camera/`：相机打开、相机参数读取、内参缩放。
- `calibration/`：棋盘格角点检测、相机标定、重投影误差计算。
- `perception/`：ArUco 检测和 marker 位姿估计。
- `control/`：视觉伺服 P 控制、RC 映射、简单滤波。
- `utils/`：YAML 读写、时间戳、日志工具。

## 开发规则

- 通用功能放在 `src/`，实验入口放在 `scripts/`。
- 不在 `src/` 中硬编码某一次实验目录。
- 新增模块后，优先让脚本通过配置文件调用。
