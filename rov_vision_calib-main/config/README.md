# config 配置文件说明

本目录保存实验运行时会读取的配置文件。这里的文件应当小、清晰、可手动检查，不保存大量实验数据。

## 文件说明

- `camera_rov_front.yaml`：前视相机硬件与采集参数，包括设备节点、分辨率、帧率、曝光、白平衡、防水舱信息。
- `calib_underwater_20_50cm_1080p.yaml`：当前默认使用的 20-50 cm、1080P 水下等效针孔标定参数。第一次标定前是模板，标定后会写入真实的 K、D 和误差。
- `aruco_config.yaml`：ArUco/AprilTag 字典、marker ID、marker 尺寸、使用的标定文件路径和检测参数。当前位姿验证阶段使用 AprilTag `DICT_APRILTAG_36h11`，但目录和 CSV 命名仍保留 `aruco`。
- `servo_config.yaml`：视觉伺服目标位置、P 控制增益、RC 中位值、限幅和滤波参数。

## 使用规则

- `config/` 里的标定文件代表“当前默认使用版本”。
- 每次实验的完整结果必须保存在 `data/`，不要只依赖 `config/`。
- 相机分辨率、曝光、增益、白平衡、对焦、防水舱位置改变后，应重新标定或至少重新验证。
